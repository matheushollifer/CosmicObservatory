# CosmicObservatory
Um observatório digital para simulações astronômicas e estudos de cosmologia. Experimentos em mecânica orbital, sistemas estelares e visualizações do universo através de programação e computação gráfica.

# 2D Solar System Simulation

![Data](https://img.shields.io/badge/Data-2_de_Novembro_de_2025-blue?style=for-the-badge)

Este projeto é uma simulação visual de um sistema solar desenvolvida em C++ utilizando a biblioteca gráfica Raylib. O sistema representa o Sol no centro da tela com oito planetas orbitando ao seu redor em trajetórias circulares, cada um com tamanho, cor, distância orbital e velocidade de translação específicos. A simulação inclui ainda a Lua orbitando ao redor da Terra, completando um modelo simplificado do nosso sistema solar. O funcionamento baseia-se em cálculos trigonométricos para determinar as posições dos corpos celestes em tempo real, atualizando os ângulos orbitais a cada frame para criar a ilusão de movimento contínuo. A aplicação oferece interatividade através do teclado, permitindo alternar entre modo janela e tela cheia, enquanto renderiza órbitas como anéis cinzentos e corpos planetários como círculos coloridos sobre um fundo negro, criando uma representação estilizada mas visualmente clara da mecânica orbital básica.

# 3D Solar System Simulation

![Data](https://img.shields.io/badge/Data-2_de_Novembro_de_2025-blue?style=for-the-badge)

Este projeto é uma simulação tridimensional de um sistema solar desenvolvida em C++ utilizando a biblioteca gráfica Raylib para renderização 3D. O sistema representa o Sol como uma esfera central amarela com oito planetas orbitando em trajetórias circulares tridimensionais, cada um com tamanho, cor, distância orbital e velocidade de translação específicos. A simulação inclui a Lua orbitando ao redor da Terra em um sistema planetário completo, com todos os corpos celestes representados como esferas tridimensionais que se movem no espaço. O funcionamento baseia-se em cálculos trigonométricos aplicados às três dimensões para determinar as posições orbitais em tempo real, atualizando os ângulos de rotação a cada frame com base no tempo decorrido. A aplicação oferece controle interativo de câmera, permitindo navegação livre pelo espaço 3D ou rotação automática ao redor do sistema, enquanto renderiza órbitas como anéis translúcidos no plano horizontal e uma grade de referência para contextualização espacial, criando uma representação imersiva da mecânica orbital em ambiente tridimensional.
